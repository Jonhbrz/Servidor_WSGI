Instalación Básica de Waitress

1 min

En esta sección aprenderás a instalar Waitress, un servidor WSGI de producción para aplicaciones Python como Flask.

1. Requisitos previos#

Asegúrate de tener instalado:

  • Python 3.10 o superior
  • pip actualizado
  • Un entorno virtual (opcional pero recomendado)

2. Crear el entorno virtual#

python -m venv venv
source venv/bin/activate      # Linux/macOS
venv\Scripts\activate         # Windows

3. Instalar Flask y Waitress#

pip install flask waitress

4. Archivo básico de ejemplo#

from flask import Flask

app = Flask(__name__)

@app.get("/")
def home():
    return "Waitress funcionando correctamente."

5. Ejecutar la app con Waitress#

waitress-serve --listen=0.0.0.0:8000 main:app

Ahora el backend está listo para producción a través de Waitress.
